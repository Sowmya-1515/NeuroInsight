Model
Technical details

Built on ResNet-50 with a custom multi-task learning framework.

Backbone

ResNet-50 Encoder

ImageNet-pretrained backbone with all layers unfrozen after the first 3 warmup epochs. Extracts 2048-dimensional feature vectors from 160×160 MRI slices.

  • Input: 160×160×3 RGB
  • Backbone: ResNet-50 (pretrained)
  • Feature dim: 2048
  • Warmup: 3 epochs frozen
  • Dropout: 0.3 throughout
Projection Head

128-dim Embedding

A 3-layer projection head maps the 2048-dim backbone output to a 128-dim L2-normalised embedding space used for FAISS retrieval.

  • Linear(2048 → 512) + ReLU + BN
  • Linear(512 → 128)
  • L2 normalisation
  • Loss: SupCon + NTXent
  • Temperature: 0.07
Classification Heads

5 Parallel Heads

Five independent classification heads operating on the 2048-dim feature vector. Severity uses Focal Loss to handle class imbalance.

  • Grade: Linear → ReLU → Dropout → Linear(2)
  • Severity: Focal Loss, γ=2.0, 3 classes
  • Size: 3 classes (small/medium/large)
  • Location: 3 classes (left/right/bilateral)
  • Tumor: 2 classes (present/absent)
Loss Function

Multi-Task Loss

Uncertainty-weighted multi-task loss (Kendall et al. 2018). Learnable log-variance parameters automatically balance all task losses during training.

  • SupCon: supervised contrastive
  • NTXent: noise-contrastive estimation
  • CE losses per classification head
  • Focal loss for severity
  • Uncertainty weighting (learned σ)

Dataset
BraTS 2020

Trained and evaluated on the Brain Tumour Segmentation (BraTS) 2020 challenge dataset.

Dataset Overview

BraTS 2020 contains multi-institutional pre-operative MRI scans of glioma patients, including HGG (high-grade) and LGG (low-grade) cases. Axial slices were extracted at 224×224 and resized to 160×160 for training.

Label Construction

Rich composite labels were derived from tumour metadata: size bins (small/medium/large), laterality (left/right/bilateral), slice position, subregion presence (necrosis/edema), and severity scores from tumour area normalisation.

Training Setup

Batch size 32, Adam optimiser with differential learning rates (backbone 10× lower than heads). Augmentation: horizontal flip, vertical flip, ±15° rotation, brightness/contrast jitter. Weighted sampler for class balance.
